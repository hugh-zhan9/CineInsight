# AI 智能增强开发计划

## Phase 1
- 补齐本地优先的标签分析客户端
- 让远程配置缺失时也可工作
- 保持现有候选审核接口不变
- 状态：已完成基础实现

## Phase 2
- 接入轻量人脸检测资源
- 增加人脸签名和聚类逻辑
- 让人脸结果参与候选生成
- 状态：已完成后端基础实现

## Phase 3
- 前端展示分析摘要
- 审核页显示人脸与视觉证据
- 增加局部刷新与状态提示
- 状态：已补充视频列表的人脸分析入口和智能搜索入口；审核页证据摘要仍待继续

## Phase 3.5
- 增加轻量自然语言搜索
- 复用标签、字幕、人脸和视频元数据索引
- 保持默认无模型下载
- 状态：已完成基础实现

## Phase 4
- 评估是否需要额外远程增强模型
- 仅作为可选，不作为默认路径
- 状态：待继续

## 开发约束
- 每一步都先补测试
- 每一步都尽量小
- 保持现有标签审核流程不被破坏
